# Capstone_1 EmporiUm Sales Territory Analysis
For this project I stepped into the role of a data analyst for EmporiUm, a student bookstore chain selling textbooks,technology, art supplies and apparel. My job was to analyze sales performance for the New Jersey territory and provide recommendations to sales manager Miami Vue. The analysis was performed using MySQL on the sample_sales database, with visualization created in Excel.


## My Sales Territory
Territory: New Jersey (In-Store) -
Region: Northeast -
Sales Manager: Miami Vue


## What I Analyzed
1. Total revenue and date range
2. Month by month revenue breakdown
3. New Jersey vs Northeast region comparison
4. Transactions and average sale by product category
5. Store performance ranking
6. Recommendation for next quarter
